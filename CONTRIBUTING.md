# Contributing to Long-Context Attention Lab

We follow a rigorous scientific approach. Please ensure:
1. **Numerical Stability:** Any new kernel must pass the stability test in `tests/test_stability.py`.
2. **Benchmark Integrity:** Performance improvements must be documented using `benchmarks/profile_scaling.py`.
3. **Mathematical Documentation:** Major architectural changes require a corresponding update in the `docs/` folder using LaTeX notation.
